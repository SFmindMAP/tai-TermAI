## üìÇ How to Install

1. Install g4f `pip install -U g4f`
2. Download lastet realease
3. Move `tai` to `/usr/local/bin` `sudo mv tai /usr/local/bin/`
4. Make it executable `sudo chmod +x tai`

ok now you can run it, just type "tai" in terminal
